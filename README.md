## Installation

```sh
npm install -g @julong/jbin
npm install --save-dev @julong/jbin
```



## Usage

```sh
jbin -t ts -p ./src/index.ts -o ./dist/index.ts
```

### --type(-t)
- ts
- js

### --path(-p)
- ./src/index.${type} (default)

### --path(-p)
- ./dist/index.js (default)
